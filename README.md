# 🌤️ Weather App

Aplicativo de previsão do tempo desenvolvido com HTML, CSS e JavaScript puro, consumindo dados de uma API externa em tempo real.
**👉 [Clique aqui para acessar o site](https://italonobrega.github.io/weather-app/)**

## ✨ Funcionalidades

* Busca de previsão do tempo pelo nome da cidade
* Integração em tempo real com a API do OpenWeatherMap
* Exibição de ícone dinâmico, temperatura, umidade, velocidade do vento e sensação térmica
* Tratamento de erros amigável (cidade não encontrada ou campo vazio)
* Layout limpo, moderno e 100% responsivo

## 🛠️ Tecnologias

* HTML5
* CSS3
* JavaScript (ES6+, async/await, Fetch API)
* OpenWeatherMap API
